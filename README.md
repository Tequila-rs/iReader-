# iReader 电子书阅读工具

一个轻量级、专注于阅读体验的Web电子书阅读器。

## 功能特性

### 📖 核心阅读功能
- 支持TXT格式电子书阅读
- 智能分页系统
- 流畅的翻页体验
- 阅读进度自动保存

### 🎨 个性化设置
- 亮色/暗色主题切换
- 字体大小调节 (12px - 24px)
- 响应式设计布局

### ✨ 交互功能
- 文本高亮标记
- 书签添加功能
- 键盘快捷键支持 (左右箭头翻页)
- 浮动工具栏

### 📱 用户体验
- 简洁直观的界面设计
- 平滑的动画过渡效果
- 即时的操作反馈

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **架构**: 纯前端实现，无后端依赖
- **兼容性**: 现代浏览器 (Chrome, Firefox, Safari, Edge)
